#### 开始
- 需要使用 MSSQL 2012 及以上版本数据库
- 需要使用VS2017及以上版本，建议使用VS2019
- 修改mvc项目下appsettings.json 中的数据库链接字符串
- 设置LyyCMS.Web.Mvc 为启动项
- 打开程序包管理器控制台，选择LyyCMS.EntityFrameworkCore, 执行 `update-database`
- 启动程序，登录用户名: admin,  密码： 123qwe
